# Third-party icon credits

Glidna's custom icon set (`src/icons.jsx`) is mostly original, drawn in-house.
Some workout/activity glyphs are adapted from permissively-licensed open-source
icon libraries. All are free for commercial use; this file preserves the required
attribution.

## Phosphor Icons — MIT License
https://github.com/phosphor-icons/core · © Phosphor Icons

Used for the exercise-category glyphs: `walk`, `run`, `bike`, `swim`, `hike`,

`tennis`, `volleyball`, `football`, `pingpong`, `carry` (suitcase-simple).
Path data transformed from Phosphor's 256×256 grid into our 24×24 viewBox.
